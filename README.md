# 🌱 佛系种地 (Zen Farm)

用真实天气种虚拟植物的佛系小游戏。

## 核心玩法

- 🌦️ **真实天气联动** — 接入 Open-Meteo API，当地天气直接影响植物生长
- 🌿 **拟真植物模拟** — 温度、阳光、风速、降水、土壤湿度影响植物形态
- 🧘 **佛系 / 硬核双模式** — 佛系模式显示所有数据，硬核模式隐藏关键信息
- 📱 **微信小游戏** — 基于 Cocos Creator 3.8 (3D)

## 植物

| 植物 | 难度 | 生长周期 |
|------|------|----------|
| 🍀 幸运草 | ★ | 7 天 |
| 🌻 向日葵 | ★★ | 30 天 |
| 🍓 草莓 | ★★★★ | 90 天 |
| 🌸 樱花 | ★★★★★ | 365 天 |

## 天气影响植物形态

- ☀️ 阳光充足 → 矮壮、叶多、叶色深
- 🏠 遮挡/温室 → 徒长（高但细弱）
- 🌬️ 大风 → 茎秆变粗、植物倾斜
- 💧 干旱 → 根系深扎
- 🌧️ 潮湿 → 根系浅、易涝

每棵植物因天气不同而独一无二。

## 地块系统

- 初始 1 个地块
- 收获一棵植物 → 解锁新地块
- 左右滑动切换地块

## 设施

- 🏠 遮雨棚 — 挡雨防涝，24 小时后过期
- 💨 除湿器 — 降低土壤湿度

## 技术栈

- **引擎**: Cocos Creator 3.8 (3D)
- **语言**: TypeScript
- **天气 API**: Open-Meteo（免费）
- **目标平台**: 微信小游戏

## 项目结构

```
assets/scripts/
├── PlantTypes.ts    # 植物配置（4种植物、阶段、物理特征）
├── Environment.ts   # 天气系统（Open-Meteo API、土壤湿度）
├── Plant.ts         # 植物模拟（生长、形态、胁迫、里程碑）
├── GameData.ts      # 游戏数据（地块、浇水、收获、设施）
├── Storage.ts       # 存档系统（localStorage）
└── ZenFarmGame.ts   # 主控 + UI（Cocos 组件）
```

## 开发

1. 用 Cocos Creator 3.8 打开 `zen-farm-cocos/` 项目
2. 浏览器预览：`localhost:7456`
3. 微信小游戏：构建发布 → 微信开发者工具
